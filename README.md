# AgriScan - AI Plant Disease Detection

## Description
AgriScan is a mobile application designed to detect plant diseases using artificial intelligence. The goal is to help farmers identify plant issues and improve agricultural productivity.

## Technologies
- Flutter (Frontend)
- Django (Backend)
- Python (Machine Learning)
- Google Colab
- Kaggle (Dataset)

## Features
- Image-based plant disease detection
- Simple and user-friendly interface
- AI-powered analysis
